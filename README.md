Personal AI Knowledge Assistant

EnteAI is a lightweight Retrieval-Augmented Generation (RAG) system that allows users to upload documents and interact with them using an AI-powered interface.

The system extracts text from documents, converts them into semantic embeddings, stores them in a FAISS vector database, and generates context-aware responses using a local LLM via Ollama.
