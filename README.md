EnteAI

EnteAI is a lightweight AI-based document assistant built using Python and Streamlit.

The application allows users to upload documents, process text, perform semantic search, and interact with a language model to retrieve relevant answers.

Features

Document upload & processing

Text chunking

Embedding generation

Vector-based search

LLM-powered responses

Simple Streamlit UI

Tech Stack

Python

Streamlit

Embeddings / Vector Store

Local LLM (Ollama)
